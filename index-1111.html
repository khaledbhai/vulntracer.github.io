About

I came to security from dust, I was a noob — which means everything I know was learned by taking things apart, breaking them, and writing down what happened. Three years in, that's still how I work.

My focus is web application and API security: cross-site scripting, file upload flaws, SQL injection, IDOR, broken authentication, SSRF, and the smaller logic errors that automated scanners miss. I write Python tooling to speed up that process, and I test everything on real, reproducible setups before I trust it.

Alongside client work on Fiverr, I run a Bengali-language YouTube channel teaching Kali Linux and cybersecurity fundamentals — because most of what I needed when I started wasn't available in my own language.
